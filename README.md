# Iranian SMS Testing Tool

A Python-based SMS testing tool developed for educational purposes, security research, and studying SMS verification systems used by Iranian online services.

## Features

* Python implementation
* Simple and lightweight
* Multiple service support
* API interaction examples
* Educational security research

## Requirements

* Python 3.10+
* requests

## Installation

```bash
git clone https://github.com/your-username/your-repository.git
cd your-repository
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
